## Короткий алгоритм розгортання
- Проведіть підготовчі дії (папка, Herd, клонування репозиторію).
- Встановіть залежності (за потреби).
- Встановіть WordPress через CLI.
- Виконайте міграцію сайту.
- Проведіть налаштування адмінпанелі та оптимізацію.
- Зберіть статику (за потреби).
- Видаліть стандартні теми (за потреби).

## 1. Підготовка
1. Створіть папку для проекту.
2. Додайте сайт у Herd або скористайтесь іншим локальним середовищем.
3. Клонуйте репозиторій проекту командою:
   ```bash
   git clone <ваш-repository-url> .
   ```
   > **Щоб репозиторій розгорнувся саме у поточній (кореневій) папці, додайте крапку в кінці команди.**

## 2. Встановіть залежності
Встановіть npm/composer залежності для коректної роботи проєкту.

## 3. Встановіть WordPress за допомогою CLI
> **Поля типу DB_NAME, SITE_URL обов'язково заповнити власними даними!**

```bash
wp core download
```

```bash
wp config create --dbname=DB_NAME --dbuser=root --dbpass='' --dbhost=127.0.0.1
```

```bash
wp db create
```

```bash
wp core install --url="SITE_URL" --title="" --admin_user="admin" --admin_password="12345" --admin_email="admin@admin.local" --skip-email
```

> **Примітка:** Ви можете оптимізувати цей процес, створивши виконуваний bash-скрипт, у якому всі ці команди будуть прописані наперед.

## 4. Міграція сайту
1. Встановіть та активуйте плагін WP Migrate Pro.
2. Виконайте міграцію сайту з Stage або Prod (перша міграція повинна бути повною).

> **Налаштування міграції буде доповнено пізніше.**

## 5. Налаштування адмінпанелі та оптимізація локального середовища
1. Вимкніть непотрібні для локальної розробки плагіни (наприклад: Redis, Cache, Perfmatters та інші).
2. За потреби, виконайте додаткові налаштування адмінпанелі для зручної роботи на локалці.

## 6. Збірка статики (за потреби)
Якщо проєкт передбачає використання збірки статики (наприклад, build для JS/CSS), виконайте цю дію згідно документації вашої теми чи плагінів.

## 7. Видаліть стандартні теми WordPress (необов'язково)
> **Цей пункт не є обов'язковим і виконується за потреби.**
```bash
wp theme delete twentytwentythree twentytwentyfour twentytwentyfive
```
